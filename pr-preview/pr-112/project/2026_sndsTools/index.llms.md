# Synthèse du projet

|  | `sndsTools`, un package R pour l’extraction de recours aux soins dans les données de santé du SNDS |
|----|----|
| **Détail du projet** | `sndsTools` est un package R conçu pour faciliter l’extraction de recours aux soins à partir des données de santé du Système National de Données de Santé (SNDS) hébergées sur le portail de l’Assurance Maladie. Il simplifie les étapes d’extraction de données du SNDS sur des données utilisées dans la majorité des études sur le SNDS. |
| **Acteurs** | Insee, Institut du Cerveau, Inria, AP-HM |
| **Résultats du projet** | Le package R `sndsTools` est en production et téléchargeable depuis GitHub . |
| **Produits et documentation du projet** | [Site de documentation du package R](https://sndstoolers.github.io/sndsTools/index.html) |
| **Code du projet** | \- Le code est disponible sur GitHub  <https://github.com/SNDStoolers/sndsTools> |
