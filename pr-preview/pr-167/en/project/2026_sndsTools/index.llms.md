## Project summary

|  | `sndsTools`, an R package for extracting healthcare utilization in SNDS health data |
|----|----|
| **Project details** | `sndsTools` is an R package designed to simplify the extraction of healthcare utilization from the Système National de Données de Santé (SNDS) health data hosted on the Health Insurance portal. It streamlines the data extraction steps from the SNDS for data used in the majority of SNDS studies. |
| **Players** | Insee, Institut du Cerveau, Inria, AP-HM |
| **Project results** | The R package `sndsTools` is in production and available for download from GitHub . |
| **Products and project documentation** | [R package documentation site](https://sndstoolers.github.io/sndsTools/index.html) |
| **Project code** | \- The code is available on GitHub  <https://github.com/SNDStoolers/sndsTools> |
